## Overview

An editorial blog built for a client using Astro and Contentful as the headless CMS. The priority was performance — every page passes Core Web Vitals with green scores across the board.

## Stack

**Astro** provides zero-JS static output. The Contentful integration pulls content at build time via the Delivery API — no client-side fetching, no FOUC, no hydration cost.

**Contentful** gives the client a familiar, polished editing experience without me managing CMS infrastructure.

## Performance

- LCP under 1.2s on desktop
- Zero render-blocking scripts
- Images served via Contentful's image transformation API with automatic WebP conversion
